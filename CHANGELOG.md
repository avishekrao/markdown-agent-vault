# Changelog

All notable public changes are tracked here.

This project uses early semantic versions. Until `v1.0.0`, breaking changes to structure or rules may happen when they make the starter kit clearer or safer.

## [v0.1.0] - 2026-05-02

First English-first starter-kit release.

### Added

- Public README with positioning, first 10-minute flow, comparison table, maturity status, and validation commands.
- `AGENTS.md`, `START_HERE.md`, `ONBOARDING.md`, and `QUICKSTART.md`.
- MIT license.
- Contribution, support, roadmap, and pull request guidance.
- GitHub issue templates for bugs, documentation fixes, examples, and method changes.
- Minimal `examples/first-session/` workflow showing an inbox note routed into a project.
- Release notes under `docs/releases/v0.1.0.md`.

### Notes

- This is not a mirror of the Russian version.
- Tool-specific compatibility claims are not included yet.
- The safest first run is in a copy, not your real vault.

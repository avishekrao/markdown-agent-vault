# Anti-memory

## Essence

Anti-memory prevents weak, old, or ambiguous material from becoming durable truth. It is a negative rule set for memory extraction and current-state rebuilds.

## Do Not Promote Automatically

Do not promote these items to current truth without evidence and authority:

- a meeting participant's opinion;
- a client wish;
- a draft plan;
- an old task list;
- a confident agent summary;
- repeated wording without independent confirmation;
- a hypothesis that sounds plausible;
- a historical decision that may have been superseded;
- a status that has not been verified recently.

## Required Checks

Before writing an important memory claim, identify:

- claim type;
- source;
- evidence;
- confidence;
- last verified date;
- write policy;
- whether the source had authority to make the claim.

## Conflict Rule

If two claims contradict each other, do not smooth the contradiction into a clean summary. Create or update a memory conflict in [conflicts](./conflicts/README.md).

# Memory Conflicts

## Essence

This directory stores unresolved contradictions in durable vault memory. A conflict is better than a beautiful summary that hides uncertainty.

## When To Create A Conflict

Create or update a conflict when:

- two sources disagree on an important fact;
- a new decision supersedes an older one but the old one still appears active;
- a meeting note contradicts a plan, context file, or current picture;
- a status changed but the source or date is unclear;
- the agent cannot safely decide which claim should govern current work.

## File Shape

Use [memory-conflict template](../../templates/memory-conflict.md).

## Next step

Resolve conflicts only with adequate source evidence or owner decision. Record status changes in [memory ledger](../ledger.md).

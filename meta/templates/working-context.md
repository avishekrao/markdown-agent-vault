# Working Context

## Essence

A compact current picture for a long-lived project or contour. This is not an archive and not a meeting summary.

## Current State

What is true now and should guide work.

## Active Decisions

Decisions that currently govern work, with links to sources.

## Open Questions

Questions that should not be treated as facts.

## Stale Or Superseded

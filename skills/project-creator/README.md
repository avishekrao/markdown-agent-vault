# project-creator

## Essence

This skill creates a new active project in `01_now/projects/`: the agent writes `plan.md`, `tasks.md`, `context.md`, `log.md`, `README.md`, and updates the relevant indexes.

The important part is not folder creation. The skill turns a request or an ongoing conversation into a usable project frame: goal, boundaries, intent lock, owner-interaction rules, milestones, quality criteria, first execution step, and navigation.

## Files

- [SKILL.md](./SKILL.md) - main operating instruction.
- [plan-authoring.md](./references/plan-authoring.md) - how to write `plan.md` autonomously.
- [starter-files.md](./references/starter-files.md) - templates and examples for the starter files.
- [worked-examples.md](./references/worked-examples.md) - end-to-end examples for both entry scenarios.
